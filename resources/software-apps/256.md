<h2>📝 资源描述</h2>

<div class="resource-description-content"><p>KernelSU 是 Android GKI 设备的 root 解决方案，运行在内核空间，为用户空间应用授予 root 权限。支持通过 overlayfs 修改 /system，并提供自定义 root 权限功能。</p><p><img src="https://panbox.cc/upload/image/2025-11-18/external-1763461251273-ysx70l.webp" alt="KernelSU 界面预览" /></p><h3>特色功能</h3><ul><li>基于内核模式运行，对用户空间应用有更强的掌控</li><li>白名单访问控制，仅授权 App 可访问 su</li><li>受限制的 root 权限，自定义 su 的权限规则</li><li>模块系统支持，可通过 overlayfs 加载自定义插件</li></ul><p>适合 Android 开发者和技术爱好者使用，适用于需要更高权限控制的场景，能够有效隐藏 Root 存在。</p></div>
